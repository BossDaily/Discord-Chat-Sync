# Discord-JS-Bot-Template
Template for creating a DiscordJS bot 

This includes 
 - Command Handler
 - Event Handler
 - ReadMD template
 - Proper git ignore
 - ENV example
 - Login
 - Dependancies
 - Folders
 
Command and event handler is the same one based off of the [Discord.JS guide](https://discordjs.guide/)
